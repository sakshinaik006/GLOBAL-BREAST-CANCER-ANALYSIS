# GLOBAL-BREAST-CANCER-ANALYSIS
Global Breast Cancer Exploratory Data Analysis (EDA)
This project performs a comprehensive exploratory data analysis on global breast cancer statistics, focusing on incidence, mortality, and the impact of healthcare access on survival rates across different continents.

Project Overview
Objective: Analyze trends in breast cancer survival rates and their correlation with screening programs and treatment access.

Key Features:

Statistical summary and outlier detection.

Regional analysis of incidence and mortality rates.

Correlation analysis between mammography coverage and survival.

Feature engineering (Case Fatality Ratio, Normalization, and Categorical Tiers).

Dataset Description
The analysis utilizes breast_cancer_by_country.csv, which includes:

Incidence Rate: New cases per 100,000 population.

Mortality Rate: Deaths per 100,000 population.

Five-Year Survival Pct: Percentage of patients surviving five years post-diagnosis.

Treatment Access Score: A metric (0-100) representing the availability of medical care.

Screening Program: Boolean flag indicating national screening initiatives.
